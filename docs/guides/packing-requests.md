# Packing Requests

Use this guide in the warehouse flow.

## 1. Open warehouse queue

1. Tap **Warehouse** in the top navigation.
2. Review cards marked **[ REQUESTED ]**.
3. Read person, location, and item list.

## 2. Pack items accurately

1. Match item names and quantities exactly.
2. Check modifiers if present.
: Example: `Shoes (size 11)`.
3. Place packed items together per request.

## 3. Mark request ready

1. Tap **Mark Ready** on the request card.
2. Wait for completion.
3. Confirm toast: **Request marked ready**.

## 4. Verify queue update

1. Confirm request disappears from warehouse queue.
2. If it remains, retry once after checking connection.

## Tips

- Pack one request at a time to reduce mix-ups.
- Double-check quantities before marking ready.
- Ask field volunteers if a modifier is unclear.
