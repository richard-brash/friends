# Recording a Request

Use this guide when someone asks for help at a location.

## 1. Open encounter screen

1. Go to **Routes** and open your active stop.
2. Tap **New Encounter**.
3. Verify the location shown at the top of the screen.

## 2. Identify the person

1. Under **Recently Seen Here**, tap a name if available.
2. If the person is new, type their name manually.
3. Confirm spelling before saving.

## 3. Add requested items

1. Tap quick item buttons (Socks, Blanket, Water, etc.).
2. Tap again to increase quantity.
3. Use **Add custom item** if needed.
4. Add a **Modifier** when details matter.
: Example: `Shoes (size 11)` or `Jacket (large)`.

## 4. Add optional notes

1. Enter observations in the **Observation** field.
2. Keep notes short and clear.

## 5. Save request

1. Tap **Save** once.
2. Wait for **Saving...** to finish.
3. Confirm the success toast: **Encounter saved**.

## Tips

- Use **Repeat Last Request** when requests are similar.
- Keep item names consistent so warehouse sorting is easier.
- Do not add private medical details in notes.
