# Troubleshooting

Quick fixes for common issues during outreach.

## App does not load data

1. Check internet connection.
2. Pull down to refresh (or navigate back and reopen page).
3. Try again after a few seconds.

## Save button does nothing

1. Confirm **Person Name** is filled.
2. Confirm at least one item is selected.
3. Wait if button shows **Saving...**.

## Wrong location shown

1. Use **Change** on the encounter screen.
2. Return to route stops.
3. Select the correct location before saving.

## Request not visible in warehouse

1. Confirm encounter was saved successfully.
2. Reopen **Warehouse** screen.
3. Check if request was already marked ready.

## Delivery action failed

1. Check connection.
2. Retry once.
3. If still failing, record details and notify coordinator.

## Active route missing

1. Go to **Routes**.
2. Open your route again.
3. Confirm route name appears in header.

## Tips

- Trust success toasts as confirmation of completed actions.
- If a button stays disabled, review required fields.
- Keep notes on paper temporarily if network is unstable, then enter them once connected.
