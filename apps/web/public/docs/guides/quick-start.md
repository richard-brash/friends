# Quick Start

This guide helps you start quickly before outreach begins.

## 1. Open the app

1. Open Friend Helper on your phone.
2. Confirm the top bar shows the app name.
3. If a route is active, check the route name in the header.

## 2. Choose your route

1. Tap **Routes**.
2. Select the route you are running.
3. Review the stop list before leaving.

## 3. Open a location

1. Tap a stop from the route screen.
2. Confirm the location name at the top.
3. Check **Needs Delivery** to see what still needs action.

## 4. Record a request

1. Tap **New Encounter**.
2. Confirm the location name shown near the top.
3. Enter a person name or tap a name under **Recently Seen Here**.
4. Select requested items.
5. Tap **Save**.

## 5. Move requests through workflow

1. In **Warehouse**, tap **Mark Ready** after packing.
2. At location, use **Deliver** or **Deliver All** to record delivery.
3. Check status badges to confirm updates.

## Tips

- Use **Repeat Last Request** for faster repeated entries.
- Watch for success toasts after save and delivery actions.
- If something looks wrong, refresh by returning to the previous screen and reopening the location.
