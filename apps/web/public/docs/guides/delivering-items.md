# Delivering Items

Use this guide when handing out packed requests in the field.

## 1. Open location delivery view

1. Open your route and select the stop.
2. In the location screen, go to **Needs Delivery**.
3. Focus on requests marked **[ READY ]**.

## 2. Deliver single items

1. Tap **Deliver** for each delivered item.
2. Use **Not Found** if the person is not at the location.
3. Confirm toast feedback after each action.

## 3. Deliver all items for a request

1. Tap **Deliver All** on the request row.
2. Read the confirmation dialog carefully.
3. Tap **Confirm** only if all items were delivered.
4. Tap **Cancel** if you are unsure.

## 4. Check final status

1. Requests move from **Needs Delivery** to **Delivered**.
2. Verify with status badges.

## Tips

- Use **Deliver All** only when all items were handed over.
- If a person is absent, use **Not Found** instead of guessing.
- Keep updates immediate to avoid forgetting details later.
